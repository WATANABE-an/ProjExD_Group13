# 横シューティングゲーム

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
* 自機を操作して「階層」を登っていく。毎階層ごとにスキルが選べる。最終階層までいくとエンディングが見れる。
* 参考URL：[サイトタイトル](https://www.hoge.com/)

## ゲームの遊び方
* 矢印キーで自機を操作し，スペースキー押下による弾丸で敵を撃つ
* 敵に当たるとゲームオーバーとなる

## ゲームの実装
### 共通基本機能
* 背景画像と主人公キャラクターの描画
* 敵は球を打ってくる　発車までは自機狙いか真っ直ぐか楽な方
* スキル選択
* 強制スクロール
* ステージの端に着いたら、スキル選んで次の階層へ
* 5階層?　楽な方を
